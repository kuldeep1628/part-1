C Programming

C programming language is the basis of all the programming languages.The key advantages of learning C Programming:

  • Easy to learn
  • Structured language
  • It produces efficient programs
  • It can handle low-level activities
  • It can be compiled on a variety of computer platforms

C Program to add two numbers

#include<stdio.h>
int main()
{
int num1,num2,sum=0;   
printf("\nEnter the numbers:");
scanf("%d%d",&num1,&num2);
sum= num1+num2;
printf("\nsum=%d",sum);
return(0);
}

INPUT :-
4
5
OUTPUT :-
Enter a numbers: 4 5
sum=9

C++ Programming

C++ language is a direct descendant of C programming language with additional features such as type checking, object oriented programming, exception handling etc. You can call it a “better C”.

C++ Program to Caluclate Sum of 2 Numbers

#include <iostream>
using namespace std;
int main()
{
int n1,n2,sum = 0;
cout << "Enter the numbers: ";
cin >> n1 >> n2;
sum=n1+n2;
cout << "Sum = " << sum;
return 0;
}


INPUT :-

10 20

OUTPUT :-
Enter the numbers: 10 20
Sum = 30

Python Programming

Python is a popular programming language. It is used for:

  • Web development (server-side)
  • Software development
  • Mathematics
  • System scripting

Python Program to add 2 numbers

# This program adds two numbers

num1 = 1.5
num2 = 6.3
# Add two numbers
sum = num1 + num2
# Display the sum
print('The sum of {0} and {1} is {2}'.format(num1, num2, sum))

INPUT :-
1.5 6.3
OUTPUT :-
The sum of 1.5 and 6.3 is 7.8

Java Programming

JAVA was developed by Sun Microsystems Inc in 1991, later acquired by Oracle Corporation. It was developed by James Gosling and Patrick Naughton. It is a simple programming language. Writing, compiling and debugging a program is easy in java. It helps to create modular programs and reusable code.
